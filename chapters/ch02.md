# 챕터 2: 우분투 설치와 환경 설정

> 다양한 방법으로 우분투를 설치하고 개발 환경을 구성한다

## 학습 목표
- 가상머신, WSL, 듀얼부트 등 설치 방법을 이해한다
- 우분투 초기 설정을 완료할 수 있다
- 패키지 업데이트와 기본 도구를 설치할 수 있다
- 개발에 적합한 환경을 구성할 수 있다

## 2.2 WSL2로 우분투 설치 (Windows)

### WSL2 설치 (Windows 10/11)

```powershell
# PowerShell 관리자 권한으로 실행

# WSL 활성화 및 Ubuntu 설치 (한 번에)
wsl --install

# 또는 특정 버전 지정
wsl --install -d Ubuntu-22.04

# 설치 가능한 배포판 목록 확인
wsl --list --online

# 설치된 배포판 확인
wsl --list --verbose
```

### WSL2 초기 설정

```bash
# 첫 실행 시 사용자 이름과 비밀번호 설정
Enter new UNIX username: student
New password: ****
Retype new password: ****

# 시스템 업데이트
sudo apt update && sudo apt upgrade -y
```

### WSL2 설정 파일 최적화

```ini
# %USERPROFILE%\.wslconfig 파일 생성 (Windows)
[wsl2]
memory=4GB          # 메모리 제한
processors=2        # CPU 코어 수
swap=2GB            # 스왑 메모리
```

## 2.3 VirtualBox로 우분투 설치

### 우분투 ISO 다운로드

```
공식 다운로드: https://ubuntu.com/download/server
추천 버전: Ubuntu 22.04 LTS Server

파일명: ubuntu-22.04.4-live-server-amd64.iso
크기: 약 1.8GB
```

### VirtualBox 가상머신 설정

```
권장 사양:
- 메모리: 2GB 이상 (서버), 4GB 이상 (데스크톱)
- 저장소: 20GB 이상 (동적 할당)
- 네트워크: NAT 또는 브리지 어댑터
- CPU: 2코어 이상
```

### 설치 과정 주요 단계

```
1. 언어 선택: English (Korean은 서버에서 인코딩 문제 발생 가능)
2. 키보드 레이아웃: Korean 또는 English(US)
3. 네트워크 설정: DHCP 자동 설정
4. 저장소 설정: 전체 디스크 사용
5. 프로필 설정: 사용자명, 서버명, 비밀번호
6. SSH 설치: Install OpenSSH server 체크
7. 추가 패키지: 필요에 따라 선택
```

## 2.4 우분투 초기 설정

### 시스템 업데이트

```bash
# 패키지 목록 업데이트
sudo apt update

# 설치된 패키지 업그레이드
sudo apt upgrade -y

# 커널 포함 전체 업그레이드
sudo apt full-upgrade -y

# 불필요한 패키지 제거
sudo apt autoremove -y
sudo apt autoclean
```

### 타임존 설정

```bash
# 현재 타임존 확인
timedatectl

# 타임존 목록 확인
timedatectl list-timezones | grep Seoul

# 한국 시간으로 설정
sudo timedatectl set-timezone Asia/Seoul

# 확인
date
# Mon Jan 15 14:30:00 KST 2024
```

### 로케일 설정

```bash
# 현재 로케일 확인
locale

# 한국어 로케일 추가 (선택사항, 서버는 영어 권장)
sudo locale-gen ko_KR.UTF-8
sudo update-locale LANG=ko_KR.UTF-8

# 영어 로케일 유지 (서버 권장)
sudo update-locale LANG=en_US.UTF-8
```

### 호스트명 설정

```bash
# 현재 호스트명 확인
hostname
hostnamectl

# 호스트명 변경
sudo hostnamectl set-hostname ubuntu-server

# /etc/hosts 파일도 업데이트
sudo nano /etc/hosts
# 127.0.1.1 ubuntu-server 추가 또는 수정
```

## 2.5 필수 패키지 설치

### 개발 도구 설치

```bash
# 빌드 필수 도구 (gcc, make, g++ 등)
sudo apt install -y build-essential

# 버전 관리
sudo apt install -y git

# 텍스트 편집기
sudo apt install -y vim nano

# 네트워크 도구
sudo apt install -y curl wget net-tools

# 시스템 모니터링
sudo apt install -y htop tree

# 압축 도구
sudo apt install -y zip unzip tar

# Python 개발
sudo apt install -y python3 python3-pip python3-venv

# 한 번에 설치
sudo apt install -y \
  build-essential git vim nano curl wget \
  net-tools htop tree zip unzip \
  python3 python3-pip python3-venv
```

### 설치 확인


## 2.6 SSH 서버 설정

### OpenSSH 서버 설치

```bash
# SSH 서버 설치
sudo apt install -y openssh-server

# 서비스 상태 확인
sudo systemctl status ssh

# 서비스 시작/활성화
sudo systemctl start ssh
sudo systemctl enable ssh
```

### SSH 설정 파일 수정

```bash
sudo nano /etc/ssh/sshd_config
```

```ini
# 주요 보안 설정
Port 22                          # SSH 포트 (변경 권장)
PermitRootLogin no               # root 직접 로그인 금지
PasswordAuthentication yes       # 비밀번호 인증 (키 설정 후 no로 변경)
PubkeyAuthentication yes         # 공개키 인증 허용
MaxAuthTries 3                   # 최대 로그인 시도 횟수
AllowUsers student               # 허용할 사용자 지정
```

```bash
# 설정 적용
sudo systemctl restart ssh
```

### 방화벽 기본 설정

```bash
# UFW 활성화
sudo ufw enable

# SSH 허용
sudo ufw allow ssh
# 또는 포트 번호로
sudo ufw allow 22/tcp

# 상태 확인
sudo ufw status verbose
```

## 2.7 쉘 환경 설정

### Bash 설정 파일 구조

```
~/.bashrc        # 인터랙티브 쉘 설정 (매번 로드)
~/.bash_profile  # 로그인 쉘 설정 (로그인 시 한 번)
~/.bash_aliases  # 별칭 모음 (별도 파일로 관리 권장)
/etc/bash.bashrc # 전체 사용자 Bash 설정
/etc/profile     # 전체 사용자 로그인 설정
```

### .bashrc 커스터마이징

```bash
# ~/.bashrc에 추가

# 색상 프롬프트 설정
PS1='\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ '

# 유용한 별칭 설정
alias ll='ls -alF'
alias la='ls -A'
alias l='ls -CF'
alias ..='cd ..'
alias ...='cd ../..'
alias grep='grep --color=auto'
alias df='df -h'
alias du='du -h'
alias free='free -h'

# 히스토리 설정
HISTSIZE=10000
HISTFILESIZE=20000
HISTCONTROL=ignoredups:erasedups

# 설정 즉시 적용
source ~/.bashrc
```

### Oh My Bash 설치 (선택사항)

```bash
# Oh My Bash 설치
bash -c "$(curl -fsSL https://raw.githubusercontent.com/ohmybash/oh-my-bash/master/tools/install.sh)"
```

## 2.8 네트워크 연결 확인

### 기본 네트워크 확인

```bash
# IP 주소 확인
ip addr show
# 또는
hostname -I

# 네트워크 인터페이스 목록
ip link show

# 기본 게이트웨이 확인
ip route show

# DNS 서버 확인
cat /etc/resolv.conf

# 인터넷 연결 테스트
ping -c 4 google.com
curl -I https://ubuntu.com
```

### 고정 IP 설정 (선택사항)

```yaml
# /etc/netplan/00-installer-config.yaml
network:
  version: 2
  renderer: networkd
  ethernets:
    ens33:
      addresses:
        - 192.168.1.100/24
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]
      routes:
        - to: default
          via: 192.168.1.1
```

```bash
# 설정 적용
sudo netplan apply
```

## 핵심 요약

- **WSL2**는 Windows에서 가장 쉽게 우분투를 시작하는 방법
- **초기 설정**: 시스템 업데이트 → 타임존 → 호스트명 순서로 진행
- **필수 패키지**: build-essential, git, curl, vim, python3는 기본으로 설치
- **SSH 서버**: 원격 접속을 위해 설치하고 보안 설정 필수
- **.bashrc 커스터마이징**으로 생산성 향상 가능
- 서버 환경에서는 **영어 로케일** 유지를 권장
