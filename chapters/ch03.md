## 3.1 쉘(Shell) 이해

### 쉘이란?

```
사용자 입력
    ↓
┌─────────────┐
│   Shell     │  ← 명령어 해석기 (bash, zsh, fish)
│  (해석/실행) │
└─────────────┘
    ↓
┌─────────────┐
│   Kernel    │  ← 실제 실행
└─────────────┘
    ↓
  결과 출력
```

### 주요 쉘 비교

| 쉘 | 특징 | 기본 설정 |
|----|------|-----------|
| **bash** | 가장 범용적, 스크립트 호환성 최고 | Ubuntu 기본 |
| **zsh** | bash 호환 + 강력한 자동완성 | macOS 기본 |
| **fish** | 현대적 UX, 문법이 달라 호환성 낮음 | - |
| **sh** | POSIX 표준, 최소 기능 | 스크립트용 |

```bash
# 현재 사용 중인 쉘 확인
echo $SHELL
echo $0

# 사용 가능한 쉘 목록
cat /etc/shells

# 기본 쉘 변경
chsh -s /bin/zsh
```

## 3.2 파일과 디렉토리 명령어

### 현재 위치와 이동

```bash
# 현재 디렉토리 출력 (Print Working Directory)
pwd
# /home/student

# 디렉토리 이동 (Change Directory)
cd /var/log          # 절대 경로
cd Documents         # 상대 경로
cd ..                # 상위 디렉토리
cd ~                 # 홈 디렉토리
cd -                 # 이전 디렉토리
```

### 파일 목록 확인

```bash
# 기본 목록
ls

# 자세한 정보 (권한, 크기, 날짜)
ls -l

# 숨김 파일 포함
ls -la

# 사람이 읽기 쉬운 크기
ls -lh

# 시간순 정렬 (최신 먼저)
ls -lt

# 재귀적으로 하위 디렉토리 포함
ls -lR

# 파일 유형 표시 (/ = 디렉토리, * = 실행파일)
ls -F
```

```
ls -la 출력 해석:
drwxr-xr-x  2 student student 4096 Jan 15 10:00 Documents
│││││││││   │ │       │       │    │             └─ 파일명
│││││││││   │ │       │       │    └─────────────── 수정 시간
│││││││││   │ │       │       └──────────────────── 파일 크기(bytes)
│││││││││   │ │       └──────────────────────────── 그룹명
│││││││││   │ └──────────────────────────────────── 소유자
│││││││││   └────────────────────────────────────── 하드링크 수
│└────────────────────────────────────────────────── 권한 (rwxr-xr-x)
└─────────────────────────────────────────────────── 파일 유형 (d=디렉토리)
```

### 디렉토리 생성과 삭제

```bash
# 디렉토리 생성
mkdir project
mkdir -p project/src/main    # 중간 디렉토리 자동 생성

# 디렉토리 삭제
rmdir empty_dir              # 빈 디렉토리만 삭제
rm -r directory              # 디렉토리와 내용 모두 삭제
rm -rf directory             # 강제 삭제 (주의!)
```

### 파일 생성, 복사, 이동, 삭제

```bash
# 빈 파일 생성 / 타임스탬프 업데이트
touch newfile.txt
touch -t 202401150900 file.txt  # 특정 시간으로 설정

# 파일 복사
cp source.txt dest.txt
cp -r src_dir dest_dir      # 디렉토리 복사
cp -p file.txt backup.txt   # 권한/시간 유지하며 복사

# 파일 이동 (이름 변경)
mv old.txt new.txt
mv file.txt /tmp/

# 파일 삭제
rm file.txt
rm -i file.txt              # 삭제 전 확인
rm -f file.txt              # 강제 삭제
```

## 3.3 파일 내용 확인

```bash
# 전체 출력
cat file.txt
cat -n file.txt             # 줄 번호 포함

# 페이지 단위로 보기
less file.txt               # q로 종료, /로 검색
more file.txt               # 위로 못 올라감

# 앞부분만 보기
head file.txt               # 기본 10줄
head -n 20 file.txt         # 20줄
head -c 100 file.txt        # 100 바이트

# 뒷부분만 보기
tail file.txt               # 기본 10줄
tail -n 20 file.txt
tail -f /var/log/syslog     # 실시간 추적 (로그 모니터링)
tail -F logfile             # 파일 재생성 시도 포함

# 파일 정보
file document.txt           # 파일 유형 확인
wc file.txt                 # 줄/단어/바이트 수
wc -l file.txt              # 줄 수만
```

## 3.4 검색 명령어

### grep - 텍스트 검색

```bash
# 기본 검색
grep "error" logfile.txt

# 대소문자 무시
grep -i "Error" logfile.txt

# 줄 번호 표시
grep -n "error" logfile.txt

# 매칭되지 않는 줄 출력
grep -v "debug" logfile.txt

# 재귀적 검색
grep -r "TODO" ./src/

# 정규식 사용
grep -E "error|warning" logfile.txt
grep -E "^[0-9]+" file.txt

# 주변 문맥 출력
grep -A 2 -B 2 "error" logfile.txt  # 전후 2줄

# 단어 단위 검색
grep -w "log" file.txt
```

### find - 파일 검색

```bash
# 이름으로 찾기
find /home -name "*.txt"
find . -name "config*"

# 파일 유형으로 찾기
find /var -type f           # 일반 파일
find /etc -type d           # 디렉토리
find /tmp -type l           # 심볼릭 링크

# 크기로 찾기
find / -size +100M          # 100MB 초과
find . -size -10k           # 10KB 미만

# 날짜로 찾기
find /var/log -mtime -7     # 7일 이내 수정
find /tmp -atime +30        # 30일 이상 미접근

# 권한으로 찾기
find / -perm 777            # 권한이 777인 파일
find / -perm /4000          # setuid 비트 설정 파일

# 찾은 파일에 명령 실행
find . -name "*.log" -exec rm {} \;
find . -name "*.txt" -exec grep -l "error" {} \;
```

## 3.5 파이프와 리다이렉션

### 리다이렉션

```bash
# 표준 출력을 파일로 저장
ls -l > filelist.txt        # 덮어쓰기
ls -l >> filelist.txt       # 이어쓰기

# 표준 에러를 파일로 저장
ls /nonexist 2> error.txt

# 출력과 에러 모두 저장
command > output.txt 2>&1
command &> output.txt       # 위와 동일

# 파일을 입력으로 사용
sort < unsorted.txt

# /dev/null로 버리기
command > /dev/null 2>&1
```

### 파이프(|)

```bash
# 명령어 출력을 다음 명령어 입력으로 연결
ls -l | grep ".txt"

# 여러 명령어 연결
cat /etc/passwd | grep "student" | cut -d: -f1,3

# 정렬과 중복 제거
cat words.txt | sort | uniq

# 프로세스 찾기
ps aux | grep nginx

# 파일 수 세기
ls | wc -l

# 상위 10개 큰 파일
du -sh /* 2>/dev/null | sort -rh | head -10
```

## 3.6 텍스트 처리 명령어

### sort, uniq

```bash
# 정렬
sort file.txt               # 알파벳순
sort -r file.txt            # 역순
sort -n numbers.txt         # 숫자순
sort -k 2 file.txt          # 2번째 컬럼 기준

# 중복 제거 (sort 후 사용)
sort file.txt | uniq
sort file.txt | uniq -c     # 중복 횟수 포함
sort file.txt | uniq -d     # 중복된 항목만
```

### cut, awk

```bash
# 특정 컬럼 추출
cut -d: -f1 /etc/passwd         # : 구분자의 1번째 필드
cut -d, -f2,3 data.csv          # CSV 2,3번째 필드
cut -c1-10 file.txt             # 1~10번째 문자

# awk - 강력한 텍스트 처리
awk '{print $1}' file.txt           # 1번째 컬럼
awk -F: '{print $1,$3}' /etc/passwd # : 구분자
awk 'NR==5' file.txt                # 5번째 줄
awk '$3 > 100' data.txt             # 조건 필터
awk '{sum += $1} END {print sum}' numbers.txt  # 합계
```

### sed - 스트림 편집기

```bash
# 치환
sed 's/old/new/' file.txt          # 첫 번째만
sed 's/old/new/g' file.txt         # 전체
sed 's/old/new/gi' file.txt        # 대소문자 무시

# 특정 줄 삭제
sed '3d' file.txt                  # 3번째 줄 삭제
sed '/pattern/d' file.txt          # 패턴 매칭 줄 삭제

# 파일 직접 수정
sed -i 's/http/https/g' config.txt

# 특정 줄 출력
sed -n '5,10p' file.txt            # 5~10번째 줄
```

## 3.7 유용한 단축키와 팁

### Bash 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+C` | 현재 명령어 중단 |
| `Ctrl+Z` | 백그라운드로 보내기 (정지) |
| `Ctrl+D` | 로그아웃 / EOF |
| `Ctrl+L` | 화면 지우기 (clear) |
| `Ctrl+A` | 줄 맨 앞으로 |
| `Ctrl+E` | 줄 맨 뒤로 |
| `Ctrl+R` | 히스토리 검색 |
| `Tab` | 자동 완성 |
| `!!` | 이전 명령어 재실행 |
| `!$` | 이전 명령어의 마지막 인자 |

### 히스토리 활용

```bash
# 히스토리 목록
history
history | tail -20

# 히스토리에서 실행
!123                    # 123번 명령어 실행
!git                    # git으로 시작하는 최근 명령어

# 히스토리 검색
Ctrl+R → 검색어 입력
```

### 명령어 치환

```bash
# 명령어 결과를 변수에 저장
TODAY=$(date +%Y%m%d)
FILES=$(ls *.txt)
echo "오늘 날짜: $TODAY"

# 백틱 방식 (구식)
TODAY=`date +%Y%m%d`
```
